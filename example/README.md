# pylectron example

# Development Installation

```sh
git clone https://github.com/shashfrankenstien/pylectron.git
cd pylectron
python3 -m pip install -e .
```

## Basic Usage

```sh
cd example
pylectron install && pylectron build
python3 example.py
```


## Advanced Usage (PyInstaller)

```sh
cd example
pylectron install && pylectron build
python3 example.py
```
